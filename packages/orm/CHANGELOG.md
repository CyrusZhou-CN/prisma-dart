# Prisma Dart v5.0.0

To install Prisma ORM for Dart v5.0.0 run this command:

```bash
dart pub add orm:^5.0.0
```

or update your `pubspec.yaml` file:

```yaml
dependencies:
  orm: ^5.0.0
```
