## v0.2.0

To install Prisma Flutter Integration v0.2.0 run this command

```bash
dart pub add orm_flutter:^0.2.0
```

Or update your `pubspec.yaml` file:

```yaml
dependencies:
  orm_flutter: ^0.2.0
```

### What's Changed

- **deps**: Upgrade the `webfetch` to `^0.1.0` version
