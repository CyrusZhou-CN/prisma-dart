## v5.1.0

To install Prisma ORM for Dart v5.1.0 run this command

```bash
dart pub add orm:^5.1.0
```

Or update your `pubspec.yaml` file:

```yaml
dependencies:
  orm: ^5.1.0
```

## Runtime & CLI

- **deps**: Upgrade the Dart SDK min version to `^3.4.0`.

## Flutter Integration

- **deps**: Reset the Dart SDK to `^3.4.0`
